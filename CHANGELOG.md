# Changelog

All notable changes to Obsyx will be documented here.

---

## v0.1.0 — Foundation

Initial public release.

### Added

- Modular Lua architecture
- Workspace policy engine
- Window policy engine
- Dwindle layout configuration
- Animation engine
- Decoration engine
- Environment configuration
- Input configuration
- System keybindings
- Screenshot utilities
- Media controls
- Production project structure

### Philosophy

Obsyx adopts a modular architecture where every subsystem has a single responsibility, making the configuration easier to maintain, extend, and reuse.
